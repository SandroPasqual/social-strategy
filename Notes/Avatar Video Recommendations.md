# Avatar Video for LinkedIn

## Recommended Approach

### Phase 1 — Start Now (Zero Setup)
- **Hedra** (hedra.com) — poză + text → video cu avatar vorbitor. Gratuit. Mișcări naturale de cap, nu doar lip-sync.
- Ideal să testezi rapid dacă formatul funcționează pentru LinkedIn.

### Phase 2 — Local (Desktop cu GPU)
- **ComfyUI** + **LivePortrait** workflow — față completă, head pose, clipit, expresii, lip-sync.
- Rulezi UI-ul pe orice laptop (e lightweight), procesarea pe desktop cu GPU.
- Open-source, comunitate mare, workflow-uri gata făcute.

### Ce să EVIT
- **SadTalker / Wav2Lip** — doar gura se mișcă pe poză statică. Arată robotic, scade încrederea.

## Hardware disponibil
1. Desktop cu placă video (neștiut cât de puternică) — pentru generare locală
2. Laptop 16GB RAM — pentru UI ComfyUI sau scripturi
3. Celeron N4500 (laptop curent) — doar pentru cloud API
